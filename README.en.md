## About this Mod

ARAD is an unofficial extension for RealTrainMod (RTM), NGTLib, and MCTE.

## Disclaimer

**The developer assumes no responsibility for any issues or damages caused by the use of this mod.**  
**AppleExtended-2.4.2 or later is required.**

## Installation

Download the mod and place it in your `mods` folder.

## How to Use

This mod enables the use of maps and adds two new blocks: **Station Blocks** and **Speed Limit Blocks**.

1. Insert a train consist item (created via ARTPE in AppleExtended) into a Station Block.
2. Select stations in order on the map to create a route.
3. The train from the starting Station Block will be summoned and will stop at each station in sequence until it reaches the final destination.

## Useful Information

* Since this mod processes each train consist every tick, you may notice a decrease in TPS.
* This mod does not recognize the concept of "rails." If a train heads in the wrong direction, the mod cannot correct it.
  * (This is because rail-tracking would require pre-loading tracks, which would cause a significant drop in TPS).
* Currently, trains spawn at the starting station and despawn at the final station. It is recommended to set your start and end points in a train depot for a more realistic experience.
